About

This "lab" is a programming playground for me to try out stuff that I learned or find interested.

You might find most of the projects here useless since my goal was never to create useful stuff. The projects here are, however, most definitely fun.

Most of the projects here are published under the public domain, with a few exceptions that is using MIT license. Please refer to the LICENSE file in each project for details. If the license for a project is not specified, you are free to assume it as the public domain.

This network is called "MrSheep NET" for no reason, it is just a random name. It may sound informal, but we just love to do so.

Services

List of projects maintained by MrSheep.

  • MrSheep NET

    We operate AS208884, a non-profit eudcational network.

    Proxy Service

    We provide proxy services, to help people bypass internet censorship. We believe that the internet should remain neutral and uncensored.
    Currently all the servers are running Xray (with v2ray-poseidon to provide an automated multiuser system.)
    If you have any spare servers(even a Raspberry Pi) and want to help censored users, please drop an email to admin@mrsheep.xyz

    IPv6 tunnel

    We provide ipv6 tunnel broker service for free (fair use) to help users without public IPv4 or IPv6 access. Just send an email to noc@mrsheep.xyz, and we'll configure the session for you.
    Note: SLA is not guaranteed.
    Please refer to as208884.net for more information

    DNS server

    DNS pollution is not acceptable, so we decide to host our own DNS servers. Currently hosting DNS server only in China to provide user a clean resolver. The bandwidth is limited, so the query time may not be satisfying.
    Since this is still a experimental project, we will collect user query log with their IP anonymized, the log will then be use to block advertising and tracking domains.
    TTL for logs is 24hours
    If you still want to try it: https://resolve.cod.edu.pl:1053/dns-query
    Subjected to AdGuard privacy policy.
